# Sci-Runner: The Scientist's Journey

A browser-based runner game that teaches the real academic path of a physicist and data scientist — from undergrad coursework to running your own lab.

🎮 **[Play it now](https://yourusername.github.io/sci-runner)**

---

## About

Sci-Runner is an educational infinite-scroller where you guide a scientist through five career stages, collecting the skills and fellowships needed to advance — while dodging the very real setbacks of academic life.

The game is designed to show anyone, in a fun and accessible way, what it genuinely takes to build a career in physics and data science: the coursework, the grants, the publications, and the resilience.

## How to play

- **Space / tap** to jump
- **Double-tap** for a double jump
- **Collect skill pills** to build your academic toolkit
- **Grab gold fellowship pills** (required from PhD onward)
- **Dodge the obstacles** — rejected papers, budget cuts, and imposter syndrome included
- Collect every required skill and grant to advance to the next career stage

## Career stages

| Level | Stage | Years | Key skills |
|-------|-------|-------|------------|
| 1 | Undergraduate | 1–4 | Calculus, Linear Algebra, Physics, Chemistry, Programming |
| 2 | Master's Degree | 5–6 | Statistics, ML Basics, Research Methods, Data Analysis, Thesis Writing |
| 3 | PhD Program | 7–11 | Deep Learning, Grant Writing, Publishing, Lab Management, Specialization |
| 4 | Postdoctoral Research | 12–15 | Teaching, Networking, Leadership, Mentorship, Innovation |
| 5 | Principal Investigator | 15+ | Lab Direction, Policy Work, Industry Collab, Tenure, Legacy Research |

## Obstacles

Every scientist faces setbacks. In Sci-Runner, those look like:

- Rival Scientist
- Rejected Paper
- Lab Failure
- Budget Cut
- Imposter Syndrome

You have 3 lives per run. You get a 90-frame window of invincibility after each hit — use it wisely.

## Fellowships & grants

From the PhD level onward, collecting skill pills alone isn't enough. You'll need to grab gold fellowship pills too:

- NSF Fellowship
- NIH Grant
- DOE Award
- Sloan Fellowship

These appear at height, so a well-timed jump is required.

## Built with

- Vanilla JavaScript and HTML5 Canvas — no frameworks, no dependencies
- Fully playable in any modern browser on desktop or mobile
- Hosted free on GitHub Pages

## Running locally

No build step needed. Just clone the repo and open `index.html` in your browser:

```bash
git clone https://github.com/yourusername/sci-runner.git
cd sci-runner
open index.html
```
